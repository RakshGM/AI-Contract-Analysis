üìñ Quick Start (Local Development)

Step 1: Clone & Setup

git clone https://github.com/RakshGM/AI-Contract-Analysis.git
cd AI-Contract-Analysis
pip install -r requirements.txt

Step 2: Configure Environment

Create .env file with your API keys:

GEMINI_API_KEY=your_gemini_key_here
PINECONE_API_KEY=your_pinecone_key_here
PINECONE_INDEX=your_index_name

Step 3: Run the Application

Option A: Web UI (Streamlit)

streamlit run app_ui.py

Access at: http://localhost:8501

Option B: REST API (FastAPI)

uvicorn api_fast_uploads:app --host 0.0.0.0 --port 8000 --reload

API Docs at: http://localhost:8000/docs
